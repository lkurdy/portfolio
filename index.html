ft_transcendence - 2023/2024

Ft_transcendence was the final 42 project. It's a single-page app created by beginner web developers, but we learnt a lot along the way. The project requires a Pong game with online and offline modes, social system, leaderboards, ranks and many more.

We were 4 working on this project, I handled the backend server and the sockets in the backend and frontend using Typescript with NestJS/React. Working as a team was the biggest challenge of this project, but also a great skill that we developed.

This project felt more professional than the ones before it, everyone was competent and doing their part. Most importantly everyone was working together till the end goal, which we are very proud of.

Bomberman - 2024

Unreal Engine 5 always fascinated me, when I finally built my first PC in 2023 I immediately started learning to use it. This little project is just a demonstration of what I learnt on my own.

Just like the original Bomberman, you place a bomb, it explodes in a plus shape and destroys the crates that it hits. If it hits a player, he dies, if it hits another bomb it detonates in a chain reaction. There are stat upgrades that spawn randomly when a crate is destroyed.

I made the character's movement in C++, the rest in Blueprints. I also made the explosion projectile and the upgrades with the Niagara system.

So_Long - 2022

So_Long is the first mini-game I made during my institution's curriculum. It was a challenge to make a graphical game using C and a third-party library (MinilibX).

The visuals were inspired by one of my favourite retro-games, Pac-Man. I learnt from this project the concept of rendering.

No monsters in this one, just some dots to eat.

Cub3D - 2022

Cub3D was also a 42 project made using the same library (MinilibX) in C, but it pushed it to a whole new level.

As you can see, this one's an FPS game, unlike So_long which is a top-down. This change in the camera position added a difficulty regarding the rendering which was solved using the raycasting technique.

Raycasting allowed me to turn a 2D map, like the one above, into a 3D perspective. We can move around the map smoothly and rotate the camera to view the walls properly from different angles.
